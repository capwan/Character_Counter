# Character_Counter
Live counter of characters,words,sentences &amp; paragraphs.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🌟 Features
- **Real-time Updates:** Counters update instantly as you type.
- **Smart Logic:** Accurately filters out extra spaces, tabs, and empty newlines.
- **UX-Focused Design:** Centralized controls for "Copy" and "Clear" operations.
- **Fully Responsive:** Seamlessly adapts to mobile, tablet, and desktop screens.
- **Optimized Performance:** Lightweight Vanilla JS with debounced input handling.

## 🛠 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Modern Flexbox/Grid layouts and responsive typography via `clamp()`.
- **Vanilla JS:** Zero dependencies for maximum speed and low memory footprint.
